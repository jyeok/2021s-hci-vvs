# 2021S HCI proejct - VVS

## Running our Project

Our project currently has no backend server, so We used temporary file server in `server/root`.

To run temporary local file server, run:

```bash
# in server/
node fileserver.js
```

To run our frontend server, run:

```bash
# in project root
yarn start
```
